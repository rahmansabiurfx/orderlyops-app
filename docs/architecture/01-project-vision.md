# Project Vision

The goal of this project is to demonstrate cloud native RAG platform designed to demonstrate MLOps, DevOps, Platform Engineering and LLMOps practices.

The goal is to enable users to interact with a Retrieval Augmented Generation(RAG) chatbot while providing a complete lifecycle for data ingestion, vector search, experimentation, evaluation, deployment, observability, security, cost optimization and improvements.

I am trying to structure this projet as close to a real world platform as possible. I will separate infrastructure, application and deployment concerns in separate repos.

## My primary goals are to demonstrate my skills in the following:
```
    1. Cloud Infrastructure
    2. Docker and Kubernetes
    3. GitOps
    4. Platform Engineering
    5. MLOps and LLMOps
    6. DevOps
    7. Site Reliability Engineering(SRE)
```
## Secondary Goals
```
    1.High availability
    2.Scalability
    3.Security
    4.Observability
    5.Disaster recovery
    6.Cost optimization
```
